# SQL-Analise
Este projeto consiste na criação e análise de um banco de dados relacional em SQL que simula um cenário real de vendas. O objetivo é praticar e demonstrar habilidades em modelagem de dados, criação de tabelas, relacionamentos entre entidades e elaboração de consultas SQL para análise de informações.

O banco de dados é composto por tabelas de clientes, produtos, pedidos e itens de pedidos, permitindo análises como volume de vendas, faturamento, comportamento de clientes, produtos mais vendidos, categorias mais relevantes e evolução das vendas ao longo do tempo. Ao longo do projeto, são utilizadas consultas básicas e intermediárias, incluindo filtros, ordenações, funções de agregação e JOINs entre tabelas.

Este projeto tem foco educacional e faz parte do desenvolvimento de portfólio para a área de Análise de Dados, reforçando conceitos fundamentais de SQL aplicados a um contexto próximo ao mercado real.
